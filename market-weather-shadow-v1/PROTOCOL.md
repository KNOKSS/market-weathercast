# 장 마감 후 전향적 Shadow Test v1

## 목적

과거 검증을 모두 소진한 `market-weather-presealed-candidate-v2`를 더 이상 과거 결과에 맞추지 않고, 실제 장 마감 순서대로 평가한다. 이 단계는 앱 기능이 아니라 연구 관측이다.

## 동결 대상

- 예측 대상: 다음 1거래일 True Range
- 부가 출력: 50%·80% 범위, 자산별 최근 252회 대비 날씨 등급
- 내부 실험값: 다음 날 하위 10% 꼬리수익 발생 확률
- 금지 출력: 상승·하락 방향, 목표 수익률, 매수·매도 판단
- 모델 학습: development 자산 16개, 2024년 말까지 87,501행
- Range: Ridge, lambda 0.001
- Tail: Logistic, lambda 0.01
- 전역 구간 보정: 69,087개의 과거 시점별 causal OOF 오차
- 동결 모델 해시: `fb89083281ccfa4a3688514b4bf305949005aa1cf84abcebdac6e31e5ea4b4a4`

전향 관측 기간에는 계수·특징·구간·등급 경계를 바꾸지 않는다. 발견된 약점은 수정하지 않고 별도 연구 후보로 기록한다.

## 시점 계약

- 미국 자산은 정규장 일봉 `t`가 완전히 확정된 뒤 `t+1` 거래일을 예측한다.
- 암호자산은 UTC 일봉 `t`가 완전히 종료된 뒤 다음 UTC 일봉을 예측한다.
- 날짜 `t`의 추론 행에는 라벨이 없다.
- 오차 보정은 `t`보다 앞서 결과가 확정된 최대 252행만 사용한다.
- 미국 마감 시점의 BTC 문맥은 같은 날짜 UTC 일봉이 아니라 `source date < t`만 허용한다.

## 원장

`forecast-ledger.jsonl`과 `settlement-ledger.jsonl`은 분리된 append-only SHA-256 해시 체인이다. 예측은 사후 수정하지 않으며 정산 행은 원래 예측 행의 해시를 참조한다.

다음 상황은 자동 거부한다.

- 같은 모델·자산·기준일 예측 중복
- 해시 체인 변조
- 미래 문맥 또는 라벨이 섞인 추론 행
- 동기화되지 않은 미국 종가
- 오래된 암호자산 캐시 또는 미완성 UTC 당일봉
- 공식 원장이 생긴 뒤 모델 재동결

## 평가 계획

- 최소 관측: 60거래일
- 권장 1차 판정: 120거래일
- 주 지표: Range MAE와 ATR-scaled 기준선의 paired 차이
- 구간: 50%·80% 포함률과 폭
- 날씨: 폭풍-고요 실제 범위 및 고변동 사건률 차이
- Tail: Brier, AUC, 자산군별 이질성. 연구용으로만 판정
- 보고 단위: 전체, 자산군, 자산, 월, 변동성 국면

60일 이전 결과는 운영 점검용이며 모델 우열 결론을 내리지 않는다.

공식 관측은 2026-06-20에 개시했다. 최초 원장과 시점 감사 내용은 `OFFICIAL_START.md`에 고정했다.
