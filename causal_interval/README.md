# 완전 인과형 적응 예측구간

과거 오차까지 해당 날짜 이전 데이터로 학습된 OOF 모델에서 생성합니다. 최종 모델을 과거에 회고 적용한 오차는 사용하지 않습니다.

- 2007~2014: 초기 expanding seed folds
- 2015~2022: 동결된 네 개 main folds
- 2023~2024: 2022년 말까지 학습한 outer validation model
- 자산별 직전 252개 인과적 오차와 전체 OOF 분위수를 사전강도 100으로 결합
- 이력이 60개 미만인 신규 자산은 글로벌 구간으로 대체하고, 60~251개는 보유 이력 수만큼만 약하게 반영

봉인 시험과 앱 공식은 사용하지 않습니다.
